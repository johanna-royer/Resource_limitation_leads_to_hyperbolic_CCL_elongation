# Resource limitation leads to hyperbolic cell cycle elongation in early metazoan development

This repository contains the code associated with my master's thesis as handed in in partial fulfillment of the degree of Master of Science in Biology at the University of Graz. 
Supervisor: Bernat Corominas-Murtra

The results of this thesis were also published as part of the following publication: Aguirre-Tamaral, Adrián, Johanna Royer, Magdalena Schindler-Johnson, Jun-Ru Lee, Daniel
R. Amor, Nicoletta I. Petridou, and Corominas-Murtra Bernat (2026). „A hyperbolic cell
cycle law for early embryonic developmental timing“. In: arXiv preprint arXiv:2605.13234.
[Preprint]. DOI: 10.48550/arXiv.2605.13234.
